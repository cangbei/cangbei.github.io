两数之和

示例:
给定 nums = [2, 7, 11, 15], target = 9
因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

class Solution:
def twoSum(self, nums: List[int], target: int) -> List[int]:
for left in range(len(nums)):
right = target - nums[left]
i = nums[left+1:]
if(right in i):
x = i.index(right)
x = x + left + 1
return [left,x]
return []

SQLSever的反弹注入

主要是利用opendatasource函数，这个函数说白了就是把在某一个数据库查询的结果传输到另一个数据库的某个表中，但是有一个限制条件，就是接受数据的表的字段数必须和查询数据的表的字段数相同。

这种注入方式适用于没有回显的，注入工具注入不进去并且过于缓慢的（所以可以说是一种骚姿势？）

select * from admin –

是可以自定义的sql语句，但是要注意在sqlsever中#不能注释，要用–才可以

这个可以说是一个小知识点，但是有要求，需要接收数据的数据库有自己的公网ip。所以可以用十分钟邮箱
